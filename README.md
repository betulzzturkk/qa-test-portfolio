# SauceDemo Testing Portfolio

This repository contains my QA/Test projects for the SauceDemo web application. It demonstrates my ability to create test plans, test cases, and bug reports with screenshots, following professional QA standards.

## Folder Structure

- **Test-Plan/**: Contains the detailed test plan for the application.
- **Test-Cases/**: Contains all test cases, including Login, Cart, and Checkout scenarios.
- **Bug-Reports/**: Contains all identified bugs during testing with detailed steps, expected and actual results, severity, and status.
- **Screenshots/**: Contains screenshots for each bug and key test scenario.

## Highlights

- Functional, negative, and exploratory testing included.
- Realistic bug reports with screenshots.
- Professional structure that showcases QA process understanding.
- This portfolio demonstrates my readiness for QA/Testing roles and my attention to detail.
